Datenschutzerklärung

Ping Pong Counter · Stand: Juni 2026

1. Überblick

Ping Pong Counter ist eine Progressive Web App (PWA) zur Zählung von Tischtennis-Punkten und Ballwechseln. Die App läuft vollständig auf deinem Gerät. Es werden keine personenbezogenen Daten erhoben, gespeichert oder weitergegeben.

2. Erhobene Daten

Die App erhebt keinerlei personenbezogene Daten. Es gibt keine Nutzerkonten, keine Registrierung und keinen Server, der Daten empfängt.

Lokal auf deinem Gerät speichert die App ausschließlich deine gewählten Einstellungen (z. B. Spielernamen, Satzanzahl) – diese verlassen dein Gerät zu keiner Zeit.

3. Kamera

Die optionale Kamera-Erkennung im Ballwechsel-Zähler greift nur auf die Kamera zu, wenn du sie aktiv einschaltest. Die Kamerabilder werden ausschließlich lokal verarbeitet und nirgendwo übertragen oder gespeichert.

4. Mikrofon

Die optionale Spracheingabe nutzt die Web Speech API deines Browsers. Sie ist nur aktiv, wenn du den „🎤 Farbe"-Button betätigst. Sprachdaten werden von deinem Betriebssystem bzw. Browser lokal verarbeitet. Es findet keine Übertragung an unsere Server statt.

5. Drittanbieter

Die App verwendet keine Tracking-, Analyse- oder Werbedienste von Drittanbietern. Es werden keine Daten an Dritte weitergegeben.

6. Hosting

Die App wird über GitHub Pages bereitgestellt. GitHub kann beim Aufruf der Seite technische Zugangsdaten (IP-Adresse, Zeitstempel) in Server-Logs erfassen. Informationen dazu findest du in der Datenschutzerklärung von GitHub.
